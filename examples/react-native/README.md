# React Native Example

This example is a minimal Expo development-build app that demonstrates the Mentra Bluetooth SDK.

## Run

```sh
npm install
npx expo prebuild
npx expo run:ios
```

or:

```sh
npm install
npx expo prebuild
npx expo run:android
```

## What It Demonstrates

- Subscribing to glasses and Bluetooth status
- Scanning for compatible glasses
- Connecting to saved/default glasses
- Connecting to simulated glasses
- Displaying text
- Clearing the display
- Listening for button and battery events
